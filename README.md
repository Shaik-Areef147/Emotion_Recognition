# 🎤 Emotion Recognition from Speech using Deep Learning

## 📌 Project Overview

This project recognizes human emotions from speech audio using Deep Learning and Speech Signal Processing techniques.

The system extracts MFCC (Mel Frequency Cepstral Coefficients) features from audio files and predicts emotions such as:

- 😊 Happy
- 😢 Sad
- 😡 Angry
- 😨 Fearful
- 😐 Neutral
- 😌 Calm
- 🤢 Disgust
- 😲 Surprised
